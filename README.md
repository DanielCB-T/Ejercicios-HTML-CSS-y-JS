# Ejercicios-HTML-CSS-y-JS
Esta actividad reúne todos los ejercicios que se dejaron de tarea, abarcando los temas fundamentales de HTML, CSS y JavaScript. El objetivo principal es integrar cada uno de esos ejercicios en un único repositorio alojado en GitHub, y dentro de este crear un archivo index.html que funcione como un menú centralizado, desde el cual se pueda acceder de manera sencilla y ordenada a cada uno de los trabajos realizados. 
Para facilitar el acceso y la visualización, el repositorio se publica mediante GitHub Pages, lo que permite que todos los ejercicios estén disponibles directamente desde internet sin necesidad de ejecutarlos en un entorno local. El contenido del proyecto incluye ejercicios básicos de estructuración con HTML, prácticas sobre el uso de enlaces e imágenes, la elaboración de tablas, la construcción de formularios, la aplicación de estilos con CSS y el desarrollo de funciones sencillas con JavaScript.
